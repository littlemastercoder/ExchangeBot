  <h2>Code of Conduct</h2>
  <p>By participating in this project, you agree to abide by our community standards. Please review our full policy inside the <code>CODE_OF_CONDUCT.md</code> file before interacting with other contributors.</p>
</div>

<div id="reporting-bugs">
  <h2>Reporting Bugs</h2>
  <p>Before opening a new issue, please search existing tickets to see if the problem has already been reported. If you find a new bug, please include:</p>
  <ul>
    <li>A clear, descriptive title.</li>
    <li>Steps to reproduce the issue locally.</li>
    <li>Expected behavior versus actual results.</li>
    <li>Your system environment details (OS, runtime versions, etc.).</li>
  </ul>
</div>

<div id="pull-requests">
  <h2>Pull Request Process</h2>
  <ol>
    <li>Fork the repository and create your feature branch from <code>master</code>.</li>
    <li>Install the project dependencies and verify your local server runs correctly.</li>
    <li>Keep your commits small, focused, and well-documented.</li>
    <li>Push your branch to your fork and submit a Pull Request (PR) to our primary repository.</li>
  </ol>
</div>
